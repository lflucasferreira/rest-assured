O que é o Rest Assured?

Biblioteca Java para testar APIs REST com DSL fluente — integra com JUnit 5, Hamcrest e Allure.

  • Sintaxe BDD: given() · when() · then()
  • Validação JSON com JsonPath e schemas
  • POJO mapping com Jackson
  • Filtros para logging e captura de rede

Stack do projeto

Java 17

Records, text blocks, modern language features.

Docker

API Petclinic + testes em containers isolados.

GitHub Actions

CI com service container e Allure no Pages.

Allure

@Epic, @Feature, @Story e anexos de rede.

Client layer

public Response getAllOwners() {
    return given()
        .spec(requestSpec)
        .when()
        .get("/owners");
}

Specs nunca chamam RestAssured.given() diretamente — use clients.

Primeiro teste

@Test
void shouldReturnAllOwnersWithValidStructure() {
    Response response = ownersClient.getAllOwners();
    ResponseValidator.assertStatusCode(response.getStatusCode(), 200);
    ResponseValidator.assertFirstArrayItemMatchesSchema(
        response.getBody().asString(), "schemas/owner-schema.json");
}

Validação de schema

  • owner-schema.json — respostas de owner
